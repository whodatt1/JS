자주 사용하는 마우스 이벤트

  • click : onclick, dbclick(더블클릭), contextmen(우클릭)
  • wheel : wheel, scroll, auxclick
  • drag and drop : drag, [dragstart, dragleave, dragover, dragenter, dragend], drop
  • focus and blur : 폼태그에 커서 이동, 커서 아웃
  • input, change : 입력된 상태, 입력된 값이 바뀌는 상태
  • mouseover(mouseout) : target element의 자식 요소에까지 영향을 미침
  • mouseenter(mouseleave) : target element에만 이벤트 적용
  • submit : 폼태그가 제출될 때
