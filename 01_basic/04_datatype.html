Javascript Data Type : 개발자 콘솔(F12)을 확인하세요

  • 숫자 : Number [정수, 실수, NaN]
  • 문자 : String [따옴표(',")로 감싼 모든것]
  • 참거짓 : Boolean [true, false]
  • 객체 : Object
  • 배열 : Array (배열객체)
  • 미정된 값 : Undefined
  • 알수 없는 존재 : Null

Data type을 알고 싶을 때 사용하는 키워드 : typeof

자바스크립트의 기본 데이터 타입: Primitive Type

  • Number
  • Boolean
  • String (원래는 참조형 데이터 타입! 특수하게 변형된 형태로 변경해서 사용)
  • 다른 프로그래밍 언어에서는 String은 참조형 데이터 타입이다

자바스크립트의 참조 데이터 타입: Reference Type

  • String
  • Object
  • Array
  • Function
  • etc..
